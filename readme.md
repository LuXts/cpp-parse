# C++ 实现的简单四则运算编译器

## 项目介绍

编译原理课设的 C++ 版本。计划在编写的同时写出思路和方法。希望看完的所有人都能凭借自己写出一个简单的四则运算编译器。

> 这玩意真的能算编译器吗？

**预计目标**

* 不带负号的四则运算
* 无GUI
* 生成AST或者直接生成逆波兰式
* 计算结果

**代码说明**

* 项目要求 C++17 以上才能正常编译
* 本人环境为 gcc 12.1.0
* 避免使用指针
